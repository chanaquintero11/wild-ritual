# Wild Ritual - Coffee Brand Website

A beautiful, responsive website for the Wild Ritual coffee brand built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Navigation Bar**: Sticky navigation with smooth scrolling to sections
- **Hero Section**: Eye-catching landing area with call-to-action button
- **About Section**: Information about the coffee brand and values
- **Products Section**: Showcase of coffee products with pricing
- **Contact Section**: Contact form for customer inquiries
- **Smooth Animations**: Scroll animations and hover effects
- **Modern Styling**: Coffee-themed color scheme with professional design

## File Structure

```
wild-ritual/
├── index.html      # Main HTML file
├── styles.css      # Styling and responsive design
├── script.js       # Interactive functionality
└── README.md       # This file
```

## Color Scheme

- **Primary Dark**: #2c1810 (Deep brown - coffee color)
- **Accent Gold**: #d4a574 (Warm gold)
- **Light Background**: #f9f7f4 (Off-white)
- **Text**: #333 (Dark gray)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize content, colors, and images as needed

## Customization

### Changing Product Information
Edit the product cards in the `Products Section` of `index.html` to update:
- Product names
- Descriptions
- Prices
- Images

### Updating Brand Information
Modify the text in the `About Section` and `Hero Section` to reflect your brand story.

### Adjusting Colors
Edit the CSS variables and color values in `styles.css`:
- Navigation background
- Accent colors
- Button colors

## Adding More Features

### Add Image Gallery
Replace placeholder images with real product photos.

### Add Shopping Cart
Integrate a backend service or e-commerce platform like Shopify or WooCommerce.

### Add Blog Section
Create a new section for blog posts and coffee brewing tips.

### Add Newsletter Signup
Add an email subscription form to your contact section.

## Deployment

You can deploy this website using:
- **GitHub Pages**: Push to the `main` branch and enable GitHub Pages in settings
- **Netlify**: Connect your GitHub repo for automatic deployments
- **Vercel**: Simple deployment with automatic updates
- **Traditional Hosting**: Upload files via FTP to any web host
